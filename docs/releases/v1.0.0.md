# CompanionVault v1.0.0

这是 CompanionVault 1.0.0 的正式发布标记。

主要能力：

- 统一的人格 Schema 与质量校验
- Hermes、通用 System Prompt、SillyTavern 角色卡编译
- 人格目录与 GitHub Pages
- CLI 安装、更新、回滚、卸载和版本锁定
- `persona-distiller` 跨 Agent 人格蒸馏 Skill
- 本地聊天记录归一化、脱敏、证据提取和人工审核门禁
- 可重复构建的人格 ZIP、Python wheel 与 SHA-256 校验文件

该文档作为工作流安全的 `v1.0.0` 标签目标，不修改 GitHub Actions 工作流文件。
