PersonaDock

安装一个统一的 persona-builder Skill，直接描述你想要的人格、选择聊天记录进行证据蒸馏，或将两者结合。Skill 会自动判断模式，生成短 SOUL、详细 Skill、已审核 Memory 和可移植 PersonaPack。

pip install persona-dock
personadock skill install --target codex --scope global
# 在 Codex 中描述人格或选择聊天记录
personadock install ./my-persona.personapack --target hermes
One Skill
自动选择创建、蒸馏、混合或优化流程。
Evidence-aware
聊天蒸馏保留来源、置信度和审核过程。
Layered
SOUL、Skill、Memory 各司其职，避免上下文膨胀。
Portable
使用可校验的 .personapack 跨智能体安装。

私有、本地、可选择公开

PersonaDock 不要求 Fork、PR 或上传人格。原始聊天、设计笔记和未审核记忆默认留在本地；公共导出只是用户主动选择的可选能力。
