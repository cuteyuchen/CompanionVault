PersonaDock

创建或蒸馏自己的 AI 人格，生成短 SOUL、按需加载的 Skill 和真实 Memory，打包后安装到 Hermes、OpenClaw 等智能体。

pip install persona-dock
personadock init ./my-persona --id my-persona --name "我的人格"
personadock pack ./my-persona
personadock install ./my-persona/dist/my-persona-0.1.0.personapack --target hermes
Local-first
人格、聊天和记忆默认留在本地。
Layered
SOUL、Skill、Memory 各司其职。
Portable
使用可校验的 .personapack 跨智能体安装。

不是人格投稿仓库

PersonaDock 不要求 Fork、PR 或公开人格。公共导出只是用户主动选择的可选能力。
