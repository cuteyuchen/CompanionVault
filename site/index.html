PersonaDock

无需安装 Python。通过系统安装脚本获取独立程序，再把统一的 persona-builder Skill 安装到 Codex、Claude 或 OpenCode。直接描述人格、选择聊天记录，或将两者结合，Skill 会自动判断创建、蒸馏、混合或优化模式。

Linux / macOS

curl -fsSL https://raw.githubusercontent.com/cuteyuchen/PersonaDock/main/install.sh | sh
personadock skill install --target codex --scope global

Windows PowerShell

irm https://raw.githubusercontent.com/cuteyuchen/PersonaDock/main/install.ps1 | iex
personadock skill install --target codex --scope global
No Python required
Release 提供 Linux、Windows 与 macOS 独立程序。
One Skill
自动选择创建、蒸馏、混合或优化流程。
Evidence-aware
聊天蒸馏保留来源、置信度和审核过程。
Portable
使用可校验的 .personapack 跨智能体安装。

私有、本地、可选择公开

PersonaDock 不要求 Fork、PR 或上传人格。原始聊天、设计笔记和未审核记忆默认留在本地；公共导出只是用户主动选择的可选能力。
